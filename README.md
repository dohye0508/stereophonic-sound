# stereophonic-sound

물리적 소리 회절 현상을 기하학적으로 분석하고 수학적으로 최적화한 실시간 3D 입체 음향 시뮬레이터입니다.

## 주요 기능
- **수평 입체감(ITD & ILD)**: Woodworth 구체 모델을 기반으로 선형 근사 및 반각공식을 활용한 실시간 양이 시간차/크기차 시뮬레이션.
- **전후방 입체감**: 2차 버터워스 저역통과 필터(8000Hz)와 선형 크로스페이드(Linear Crossfade) 보간을 통한 자연스러운 후방 음 차단 효과 구현.
- **수직 입체감**: 머리와 어깨에 의한 몸통 그늘 효과(Torso Shadowing) 수학적 모사.
- **실시간 데이터 HUD**: 내적값, 볼륨 비율, 시간차(ITD) 데이터 실시간 시각화.
- **사용자 설정**: ESC 키 조작을 통한 마스터 볼륨 설정 제어.
